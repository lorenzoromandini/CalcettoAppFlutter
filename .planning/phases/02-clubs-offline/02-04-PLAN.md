  <action>
    1. Create lib/core/providers/offline_status_provider.dart:
       - offlineStatusProvider: StreamProvider<bool>
         - Watches connectivityService.onConnectivityChanged
         - Maps to true/false (true = offline)
         - Debounce slightly to avoid flicker (300ms)
       - isOfflineProvider: Provider<bool>
         - Synchronous access to current state
         - Use for conditional UI (disable buttons)
       - offlineBannerProvider: StateProvider<bool>
         - Controls visibility of offline banner
         - True when offline and should show indicator
       - lastOnlineAtProvider: StateProvider<DateTime?>
         - Track when we last had connection
         - For showing "last updated" info
       - cacheAgeProvider: Provider<Duration?>
         - Computed from lastOnlineAt
         - Shows how stale the cache is
    
    2. Create lib/core/widgets/offline_indicator.dart:
       - OfflineIndicator ConsumerWidget
       - Watches isOfflineProvider
       - Returns MaterialBanner when offline:
         - Content: Row with:
           - Icon(Icons.cloud_off, color: Colors.orange)
           - SizedBox(width: 8)
           - Expanded Text: "You're offline. Showing cached data."
         - Actions: [
           TextButton(
             onPressed: () { /* dismiss handled by provider */ },

           ),

       - Styling:

       - Position:
         - Place in Scaffold's body above content
         - Or use as persistent footer in AppBar
         - Recommended: MaterialBanner at top of screen
       - Animation:
         - AnimatedSwitcher for enter/exit
         - Fade + slide down animation
         - Duration: 300ms
       - Cache age display (optional):
         - Subtitle text: "Last updated X minutes ago"
